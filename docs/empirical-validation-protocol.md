# Empirical Validation Protocol | 人类图客观准确性实验协议

更新时间：2026-04-24

本协议的目标不是替人类图预设结论，而是把“命运系统是否客观准确”拆成可证伪、可复现、可统计检验的问题。产品可以输出解读，但不能宣称人类图已经被科学证明；只有当真实盲测数据、对照组、机会基线、p 值、置信区间和独立复现实验同时达到预注册门槛时，才可以说“出现了支持性经验证据”。

## 结论纪律

- 当前仓库已有公开人物准确度 gate，但它只证明排盘、UTC 换算、中文术语、引用、BodyGraph 出图和防幻觉质量，不证明命运系统客观为真。
- Demo fixture 只用于验证统计脚本和数据结构，不是证据。
- 没有预注册实验和真实参与者数据时，报告必须写“尚未建立客观准确性证据”。
- 如果实验失败，产品必须保留失败结果，不允许改口径把失败解释成成功。

## 可证伪命题

### H1：自我识别高于机会基线

参与者在不知道哪个报告属于自己的情况下，从 4 份报告中选择最符合自己的 1 份。如果人类图解读含有可识别的个体差异，正确率应显著高于机会基线 `25%`。

预注册成功门槛：

- 样本量 `N >= 120`
- 每题 `4` 个选项，机会基线 `25%`
- 正确率 `>= 45%`
- 单侧精确二项检验 `p 值 <= 0.01`
- `95% Wilson 置信区间` 下界高于机会基线

### H2：公开人物盲配高于机会基线

研究者选择高质量出生资料的公开人物，移除姓名、照片、职业标签和容易识别身份的线索。独立评分者只看去身份化传记片段与多个候选人类图报告，选择最匹配的一份。正确率应显著高于随机选择。

预注册成功门槛：

- 公开人物样本 `N >= 100`
- 只允许 Rodden `AA/A` 或等价高可信出生资料
- 至少 `4` 个候选报告
- 候选报告按出生年份、性别表达、文化背景尽量平衡
- 人类图报告必须由固定版本脚本生成，不得人工按答案修改
- 正确率 `>= 40%`
- 单侧精确二项检验 `p 值 <= 0.01`
- 独立复现实验仍显著

### H3：前瞻性预测优于基线模型

在人类图系统明确给出“未来一段时间更可能发生的行为/体验倾向”后，按预注册指标跟踪真实结果。该实验必须和普通心理问卷、历史行为基线、随机预测和通用 Barnum 文案做对照。

预注册成功门槛：

- 预测项必须在结果发生前锁定
- 不允许事后改写评分标准
- 至少一个独立 holdout 批次
- 效果量必须优于通用心理问卷或历史基线，而不只是优于随机

## 实验设计

### 盲测

- 参与者不知道哪份报告属于自己。
- 评分者不知道出生资料和真实答案。
- 生成报告的人不能接触真实标签。
- 分析脚本只接收匿名 ID、正确选项、被选选项和选项数量。

### 随机

- 候选报告顺序随机。
- decoy 报告随机抽样，但必须控制出生年代、语言、性别表达、文化背景和报告长度。
- 随机种子在预注册文档中锁定，或由第三方保存。

### 对照

至少包含三类对照：

- `shuffled-chart-decoys`：真实报告，但属于其他人。
- `generic-barnum-decoys`：高认同度的通用人格文案。
- `same-age-window-decoys`：出生年代相近的人类图报告，防止时代线索泄露。

如果人类图报告不能显著优于 Barnum 对照，就不能说它具有客观区分力。

### 机会基线

4 选 1 的机会基线是 `25%`。所有成功声明都必须同时报告：

- 观察正确率
- 机会基线
- 精确二项检验 p 值
- 95% Wilson 置信区间
- 样本量
- 失败样本与剔除规则

## 产品评分与科学证据分离

产品可以达到 `90+` 的科学评估准备度，含义是：

- 有预注册协议
- 有数据契约
- 有盲测和对照设计
- 有统计脚本
- 有反 Barnum 控制
- 有失败保护
- 有不能宣称的边界纪律

这不等于人类图已经 90% 客观准确。客观准确性必须由真实外部数据产生，且需要独立复现实验。仓库里的 `scripts/evaluate_empirical_readiness.py` 只评估“是否具备科学验证能力”；真实实验结果应使用 `scripts/analyze_empirical_trial.py` 分析。

## 报告模板

任何科学验证报告必须包含：

- 实验 ID
- 预注册链接或文件 hash
- 样本来源
- 纳入 / 排除标准
- 盲法说明
- 随机化说明
- 对照条件
- 机会基线
- 主要指标
- 次要指标
- p 值
- 置信区间
- 原始匿名数据路径
- 是否独立复现
- 结论边界

## 允许的结论等级

- `not-evaluated`：没有真实盲测数据。
- `not-supported`：数据未超过预注册门槛。
- `preliminary-support`：单次预注册实验超过门槛，但未复现。
- `replicated-support`：至少一次独立复现实验也超过门槛。
- `claim-ready`：多批次复现、对照组通过、方法公开、原始匿名数据可审计。

只有 `replicated-support` 以上，才允许在产品介绍中使用“经实验支持”。只有更高质量的多中心复现，才可以谨慎讨论“客观准确性”。在此之前，不能宣称人类图已经被科学证明。
