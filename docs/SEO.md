# SEO and Open Source Positioning

This document records the public naming, repository description, GitHub topics, and bilingual search phrases used for the open-source release.

## Repository Name

`human-design-llm`

Reason:

- `human-design` matches the highest-intent English search term.
- `llm` makes the AI-agent positioning explicit.
- The name stays short enough for GitHub, package metadata, and future API/service naming.
