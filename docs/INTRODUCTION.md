This file contains copy-ready descriptions for GitHub, package registries, newsletters, social posts, and external collaborators.

## One-Line English

Human Design LLM is an open-source AI toolkit for Human Design BodyGraph calculation, structured readings, career analysis, relationship comparison, timing context, and source-traceable LLM prompts.

## 一句话中文

Human Design LLM 是一个开源人类图 AI 工具库，覆盖 BodyGraph 排盘、结构化解读、职业深读、关系合盘、时机分析和可追溯引用的 LLM 产品包。

## Short English Introduction

Human Design LLM helps developers build Human Design AI assistants and BodyGraph interpretation agents. It calculates Human Design charts from birth data, converts the result into a stable JSON schema, then generates source-backed readings and LLM-ready product packages with prompts, context blocks, citations, follow-up questions, delivery depth, and session state.

Unlike prompt-only Human Design tools, this project separates chart calculation, knowledge references, interpretation logic, BodyGraph rendering, and evaluation gates. It is designed for AI products that need grounded chart facts instead of generic spiritual copy.

## 中文短介绍

Human Design LLM 帮助开发者构建人类图 AI 解读助手和 BodyGraph 解释 Agent。它会根据出生资料计算人类图，把结果转换成稳定的 JSON schema，再生成带知识卡来源的结构化解读和 LLM 产品包，包括 prompts、上下文块、引用、后续追问、输出深度和会话状态。

它不是单纯的 prompt 模板，而是把排盘、知识库、解读逻辑、BodyGraph 出图和评测门槛拆开维护，适合需要“先有盘面事实，再让大模型解读”的 AI 产品。

## Long English Introduction

Human Design LLM is an LLM-native product layer for Human Design. It wraps chart calculation, BodyGraph data modeling, interpretation references, source citations, focus-aware answer generation, and runtime-ready skill packaging into one Python repository.

The core output is not just a markdown reading. The project can produce a full LLM product package containing chart facts, reading sections, source references, assistant instructions, context blocks, answer citations, suggested follow-ups, delivery depth, and session state. This makes it suitable for AI agents, API backends, prompt orchestration systems, and local Codex/OpenAI-style skills.

The current release supports single-chart readings, career deep readings, relationship comparison, timing/transit context, birth-time uncertainty analysis, and template-based BodyGraph SVG rendering. The product is tested with pytest, smoke suites, narrative evaluations, and a V2 scorecard.

## 中文长介绍

Human Design LLM 是一个面向大模型的人类图产品层。它把人类图排盘、BodyGraph 数据结构、知识卡引用、来源追踪、焦点化回答生成和 runtime skill 打包在同一个 Python 仓库里。

这个项目的核心输出不只是 Markdown 解读，而是一份完整的 LLM 产品包：包含 chart 事实、阅读章节、来源引用、助手指令、上下文块、回答引用、推荐追问、输出深度和会话状态。因此它适合 AI Agent、API 后端、prompt orchestration、Codex/OpenAI 风格技能，以及任何需要结构化人类图事实的大模型应用。

当前版本支持单盘解读、职业深读、关系合盘、时机 / transit 分析、出生时间不确定性采样，以及模板驱动的 BodyGraph SVG 出图。产品质量通过 pytest、smoke suite、narrative eval 和 V2 scorecard 做回归验证。
